## Overview

Holideplanning simplifies the entire trip planning process — from destination research to day-by-day itinerary creation and budget management.

## Key Features

- **Destination Discovery**: Browse curated destinations with local tips, best seasons, and budget estimates
- **Itinerary Builder**: Drag-and-drop day planner with maps, timing, and activity suggestions
- **Budget Tracker**: Set trip budgets, track expenses by category, and get spending insights
- **Collaborative Planning**: Share trips with travel companions and plan together in real-time
- **Packing Lists**: Smart packing suggestions based on destination, weather, and trip duration

## Technical Highlights

- Next.js with server-side rendering for fast, SEO-friendly destination pages
- Interactive maps with route optimization between activities
- Responsive design optimized for planning on both desktop and mobile
- PostgreSQL database for reliable trip and user data storage
