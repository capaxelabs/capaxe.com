## Overview

Vizuals is a platform built for designers, photographers, and creative agencies who need a better way to organize and present their visual work.

## Key Features

- **Asset Library**: Upload, tag, and organize creative assets with smart folders and search
- **Portfolio Generator**: Create polished, shareable portfolios from your asset library in minutes
- **Mood Boards**: Collaborative mood board builder for client briefs and creative direction
- **Client Presentations**: Password-protected galleries with feedback and approval workflows
- **Brand Kits**: Centralized brand assets — logos, colors, fonts — shared across the team

## Technical Highlights

- Image optimization pipeline with automatic format conversion and responsive sizing
- AWS S3 integration for scalable, reliable asset storage
- Real-time collaboration on mood boards and portfolios
- Clean, visual-first UI designed for creative professionals
