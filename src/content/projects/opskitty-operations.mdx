## Overview

OpsKitty gives small engineering teams the operational visibility they need — deployment logs, status pages, and incident tracking — without the cost and complexity of enterprise tools.

## Key Features

- **Deployment Tracker**: Automatic deployment logs from GitHub, GitLab, and CI/CD pipelines
- **Status Page**: Public-facing status page with real-time uptime monitoring
- **Incident Management**: Create, track, and resolve incidents with timeline and post-mortems
- **Alerting**: Webhook and email alerts for downtime, failed deployments, and anomalies
- **Team Dashboard**: At-a-glance view of system health, recent deployments, and open incidents

## Technical Highlights

- Real-time data updates with Redis pub/sub for instant status changes
- Webhook integrations with GitHub Actions, Vercel, and Netlify
- Lightweight agent for server-side monitoring without heavy resource usage
- PostgreSQL for audit trail and historical data
