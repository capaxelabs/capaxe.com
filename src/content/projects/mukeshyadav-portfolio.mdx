## Overview

A personal brand website for a full-stack developer, designed to establish technical credibility and attract freelance clients and collaboration opportunities.

## Key Features

- **Performance-First**: Sub-second load times with static generation and edge deployment
- **Dark Mode**: System-aware theme switching with smooth transitions
- **Blog Integration**: MDX-powered blog for technical writing and thought leadership
- **Responsive Design**: Mobile-first layout that works perfectly on all screen sizes
- **SEO Optimized**: Structured data, Open Graph tags, and sitemap generation

## Technical Highlights

- Built with Next.js and TypeScript for type-safe development
- Tailwind CSS for utility-first styling with custom design tokens
- Deployed on Vercel with global CDN for instant page loads
- Integrated analytics for tracking visitor engagement
