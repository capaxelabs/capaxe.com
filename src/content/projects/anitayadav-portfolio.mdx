## Overview

A personal portfolio website built with a focus on clean design, fast performance, and professional presentation.

## Key Features

- **Modern Design**: Clean, minimal layout with thoughtful typography and spacing
- **Project Showcase**: Visual gallery of work with descriptions and links
- **Contact Integration**: Direct contact form with email notifications
- **Dark Mode**: System-aware theme switching
- **Mobile-First**: Fully responsive across all devices

## Technical Highlights

- Built with Next.js for static generation and optimal performance
- Tailwind CSS for consistent, maintainable styling
- Deployed on Vercel with automatic HTTPS and global CDN
- Structured data for improved search engine visibility
